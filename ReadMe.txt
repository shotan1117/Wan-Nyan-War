〇操作方法
・左スティック：プレイヤ移動
・右スティック：カメラ移動
・RBボタン：骨（弾）発射（長押し可）
・LBボタン：煙玉（ボム）発射（10秒のインターバルが存在）
・Bボタン：チュートリアル時に説明表示、メッセージ送り

〇独自の実装部、PR
・プレイヤーはそれぞれ左画面、右画面にてそうさを行う
・ショットだけではなくボムも追加
・敵の種類追加
・BGM、SE追加
・WAVEごとに敵の生成パターン変化
・チュートリアルに説明追加
・エフェクト追加
・勝敗によって勝った側がメインの一枚絵をエンディングで見れる（引き分けの場合もある）
